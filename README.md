## Head Breeze
Head Breeze is your modern note taking app that helps you manage your daily tasks and workflow in an efficient and modern way

## Tech Stack
- React JS 
- JavaScript
- Vanilla CSS  

## Features
- Create Note
- Edit Note
- Archive Note
- Trash Note
- Change Note Color
- User Details Page
- Frontend Authentication
