## Head Breeze
Head Breeze is a modern note taking app that helps you create and manage your notes efficiently with ease 

## Tech Stack
- React JS 
- JavaScript
- Vanilla CSS  

## Features
- Create Note
- Edit Note
- Archive Note
- Trash Note
- Delete Note
- Change Note Color
- User Details Page
- Frontend Authentication
